# Guía de Imágenes de Productos - Ideart

## Ubicación de las Imágenes

Todas las imágenes de productos deben colocarse en la carpeta:
```
public/images/productos/
```

## Nombres de Archivos Requeridos

Para que el sitio web funcione correctamente, necesitas agregar las siguientes imágenes con estos nombres exactos:

### Identidad Visual y Gran Formato
- `gigantografias.jpg` - Imagen de gigantografías o pancartas
- `rotulos.jpg` - Imagen de rótulos o letreros
- `rotulacion-vehicular.jpg` - Imagen de vehículo rotulado

### Oficina y Documentación
- `sellos.jpg` - Imagen de sellos automáticos o manuales
- `credenciales.jpg` - Imagen de carnets o credenciales PVC
- `tarjetas-presentacion.jpg` - Imagen de tarjetas de presentación

### Artículos Promocionales
- `jarros-personalizados.jpg` - Imagen de jarros o tomatodos sublimados
- `esferos.jpg` - Imagen de esferográficos personalizados
- `stickers.jpg` - Imagen de stickers o etiquetas

## Especificaciones Técnicas

- **Formato**: JPG o PNG
- **Tamaño recomendado**: 800x600 píxeles (mínimo)
- **Peso máximo**: 500KB por imagen (para carga rápida)
- **Orientación**: Horizontal (landscape) preferiblemente

## Ejemplo de Estructura

```
public/
  └── images/
      ├── logo.jpg (ya existe)
      └── productos/
          ├── gigantografias.jpg
          ├── rotulos.jpg
          ├── rotulacion-vehicular.jpg
          ├── sellos.jpg
          ├── credenciales.jpg
          ├── tarjetas-presentacion.jpg
          ├── jarros-personalizados.jpg
          ├── esferos.jpg
          └── stickers.jpg
```

## Notas Importantes

1. Los nombres de archivo deben ser EXACTAMENTE como se indican arriba (incluyendo guiones y minúsculas)
2. Si no tienes una imagen específica, puedes usar una imagen temporal con el mismo nombre
3. Las imágenes se mostrarán automáticamente en las tarjetas de servicio de la página principal
4. Puedes reemplazar las imágenes en cualquier momento sin modificar el código

## Optimización de Imágenes

Para mejor rendimiento, puedes optimizar tus imágenes usando:
- [TinyPNG](https://tinypng.com/) - Compresión de imágenes
- [Squoosh](https://squoosh.app/) - Optimización avanzada
